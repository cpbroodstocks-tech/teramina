# Teramina MVP

## Features

- Vite
  - html minification, EJS template
  - import aliases
  - CSS / preprocessors / modules [support](https://vitejs.dev/guide/features.html#css)
- React

## Usage

1. To start this project locally:

   ```sh
    git clone git@github.com:teramina-io/mvp-frontend.git
    yarn install
    yarn dev
   ```

2. Change the name, description, author, license, links in `package.json`.
3. Check out the available scripts in `package.json` or execute `yarn run`.

## Aliases

To add import aliases, edit these two:

- `path` in `vite.config.js`
